# rnWebRTC

## Configuration (Works on iOS & Android)
**react: 16.8.3**

**react-native: 0.59.4**

**react-native-webrtc: 1.69.1**

## Install
**Android**

-  ```git clone "url"```
-  ```cd rnWebRTC```
-  ```npm i```
-  ```react-native run-android```

**iOS**

-  ```git clone "url"```
-  ```cd rnWebRTC```
-  ```npm i```
-  open Xcode
-  on top left select *iPhone* from *Device* and click the play button
